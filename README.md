# Industrial-Portal-Automation-Agent
A robust Selenium-based automation framework designed to navigate complex web portals. Originally developed to automate data retrieval from maintenance and planning interfaces.
Business Value: Reduces manual data entry by 90%, ensuring 100% data integrity for SAP work order scheduling.

AI Pivot: This repository serves as the Action Layer for future Agentic AI integration, allowing an LLM to perform physical tasks on web interfaces via autonomous navigation.

Tech Stack: Python, Selenium, Webdriver Manager.
